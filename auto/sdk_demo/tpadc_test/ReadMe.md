## 功能介绍
此demo用于验证touchpad功能是否正常。

## 使用说明
默认把测试bin安装到rootfs中，可以直接在系统中执行tpadc_test即可。

================Usage================
tpadc_test 0      means: get /dev/event0 data
tpadc_test 1      means: get /dev/event1 data
tpadc_test 2      means: get /dev/event2 data
================usage================
